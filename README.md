# Vinayak Hattangadi - Professional Portfolio Website

## HTML Version

This is a standalone HTML/CSS/JavaScript version of the portfolio website.

## Files Included

- `index.html` - Main HTML file with all content
- `styles.css` - Complete CSS styling
- `script.js` - JavaScript for smooth scrolling and interactions

## How to Use

1. **Local Preview**: Simply open `index.html` in any web browser
2. **Deploy**: Upload all three files to any web hosting service
3. **Customize**: Edit the HTML file to update content, CSS for styling changes

## Customization Guide

### Updating Content

All content is in `index.html`. Search for the section you want to edit:
- Hero section: Search for `class="hero-section"`
- Experience: Search for `class="experience-section"`
- Skills: Search for `class="skills-section"`
- Certifications: Search for `class="certifications-section"`
- Contact: Search for `class="contact-section"`

### Changing Colors

Edit `styles.css` to change the color scheme. Key colors:
- Background: `#0a0a0a`
- Cards: `#1a1a1a`
- Borders: `#333333`
- Text primary: `#ffffff`
- Text secondary: `#c0c0c0`

### Adding Google Fonts

Add this line in the `<head>` section of `index.html`:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

## Deployment Options

### GitHub Pages
1. Create a new repository
2. Upload all three files
3. Go to Settings > Pages
4. Select main branch and save

### Netlify
1. Drag and drop the folder to netlify.com/drop
2. Your site is live!

### Traditional Hosting
1. Upload via FTP to your web host
2. Ensure all three files are in the root directory
